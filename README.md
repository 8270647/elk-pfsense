# elk-pfsense
A small collection of Logstash configurations to support pFsense 2.4.3 and dashboard visualizations within Kibana.

![alt text](https://github.com/8270647/elk-pfsense/blob/master/kibana-dashboard-visu.JPG)

# Versions

- Ubuntu Server 18.04
- pFsense 2.4.3
- Elastic 6.3.0
- Logstash 6.3.0-1
- Kibiana 6.3.0
